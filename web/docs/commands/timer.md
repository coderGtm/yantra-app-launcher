# timer

The `timer` command is used to set a countdown timer using your device's default timer app.

## Syntax
```
timer [length] [message]
```

## Usage
- `timer`: Launches your default timer app.
- `timer <length>`: Sets a timer for the specified duration in seconds (1 to 86400).
- `timer <length> <message>`: Sets a timer with a custom message.

## Example
```
timer 60
timer 3600 Take out the trash
```
