# bluetooth

The `bluetooth` command is used to toggle the Bluetooth state on your device.

## Syntax
```
bluetooth [state]
```

## Usage
- `bluetooth on` or `bluetooth 1`: Turns Bluetooth on.
- `bluetooth off` or `bluetooth 0`: Turns Bluetooth off.

!!! note
    On newer Android versions, the launcher might be restricted from toggling Bluetooth directly due to system security policies. In such cases, it will inform you about the restriction.
