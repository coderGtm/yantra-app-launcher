# music

The `music` command allows you to control local music playback on your device.

## Syntax
```
music [play/pause/prev/next]
```

## Usage
- `music`: Toggles the current music playback state (Play/Pause).
- `music play`: Resumes music playback.
- `music pause`: Pauses music playback.
- `music next`: Skips to the next track in the queue.
- `music prev`: Skips back to the previous track.

!!! info
    This command controls the active media session on your device.
