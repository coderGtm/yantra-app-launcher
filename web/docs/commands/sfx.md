# sfx

The `sfx` command allows you to play sound effects directly from the terminal.

## Syntax
```
sfx [sound_effect]
```

## Usage
- `sfx`: Plays the default sound effect.
- `sfx <sound_effect_name>`: Plays a specific sound effect that has been added to the internal storage via settings.

!!! tip
    You can manage and add new sound effects in the Launcher settings.
