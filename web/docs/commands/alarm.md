# alarm

The `alarm` command sets an alarm using your device's default alarm app.

## Syntax
```
alarm [time] [message]
```

## Usage
- `alarm`: Opens the list of alarms in your default alarm app.
- `alarm <time>`: Sets an alarm for the specified time (24-hour format, e.g., `14:30`).
- `alarm <time> <message>`: Sets an alarm with a custom message.

## Example
```
alarm 18:30
alarm 07:00 Wake up!
```
