# community

The `community` command connects you with other Yantra Launcher users and the developer.

## Syntax
```
community [discord/reddit]
```

## Usage
- `community`: Opens the official Discord server for Yantra Launcher.
- `community discord`: Navigates to the Discord server.
- `community reddit`: Navigates to the Yantra Launcher subreddit.
