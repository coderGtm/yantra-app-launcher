# info

The `info` command launches the system settings page for a specified application.

## Syntax
```
info [-p] [app/package name]
```

## Usage
- `info <app_name>`: Locates the app by its display name and opens its settings.
- `info -p <package_name>`: Opens the settings for the app with the specified package name.

## Example
```
info Chrome
info -p com.android.chrome
```
