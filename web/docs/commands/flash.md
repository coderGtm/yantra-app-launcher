# flash

The `flash` command toggles your device's flashlight.

## Syntax
```
flash [on/off | 1/0]
```

## Usage
- `flash`: Toggles the current state of the flashlight.
- `flash on` or `flash 1`: Turns the flashlight on.
- `flash off` or `flash 0`: Turns the flashlight off.

## Example
```
flash on
flash
```
