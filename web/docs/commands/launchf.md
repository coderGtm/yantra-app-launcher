# launchf

The `launchf` command stands for Fuzzy Launch. It uses a fuzzy search algorithm to find and launch the app that best matches your input.

## Syntax
```
launchf [approx app name]
```

## Usage
- `launchf <string>`: Searches for the app name that most closely matches the given string and launches it.

## Example
```
launchf tube
```
This might launch the YouTube app if it's the closest match.
