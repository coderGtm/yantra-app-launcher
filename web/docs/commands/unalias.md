# unalias

The `unalias` command is used to remove an existing alias from your session.

## Syntax
```
unalias alias_name
```

## Usage
- `unalias <alias_name>`: Removes the specified alias.
- `unalias -1`: Removes all custom aliases and resets to default.

## Example
```
unalias cod
```
This will remove the alias `cod` if it was previously set.
