# sysinfo

The `sysinfo` command displays detailed system information in a visual format, similar to the popular `Neofetch` tool on Linux.

## Syntax
```
sysinfo | sysinfo -component
```

## Usage
- `sysinfo`: Displays a summary of your device's system information along with an ASCII art logo.
- `sysinfo -<component>`: Can be used to fetch specific system details.

!!! tip
    You can customize the ASCII art displayed by the `sysinfo` command in the Launcher settings.
